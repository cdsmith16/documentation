# Server to Server Native Integration

## Overview
The TripleLift Server to Server integration is compliant with Native 1.2 Specs. This defines the structure for bid (combination of price, identifier, assets, etc.) request format and bid response format. This documentation will highlight items required by our Server to Server Native integration. Below you'll see the typical structure of a bid request and bid response object.

## Endpoint
The Server to Server integration is initiated by a bid request from the client's server to this endpoint: "http://tlx.3lift.com/s2sn/auction". TL will provide supplier id, which will be appended as query string in requests.
